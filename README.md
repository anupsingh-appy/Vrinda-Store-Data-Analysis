# Vrinda Store Data Analysis

## Project Overview
This project analyzes sales performance, customer behavior, order trends, and channel performance for Vrinda Store using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualization
- Dashboard Creation

## Key Insights
- Women contribute 64% of total sales
- Adult women are the dominant customer segment
- Amazon is the leading sales channel (35%)
- Maharashtra, Karnataka, and Uttar Pradesh are top-performing states
- 92% order delivery success rate

## Files Included
- Raw sales dataset
- Cleaned and analyzed Excel workbook
- Business presentation/report

## Business Recommendations
- Focus marketing on adult female customers
- Expand in underperforming states
- Improve marketplace optimization on Amazon, Myntra, and Flipkart
